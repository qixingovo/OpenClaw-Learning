## 🔥 三步点火流程

经过多次实践，我们总结出了最稳定可靠的 OpenClaw 启动流程：

### 第 1 步：🔄 环境净化
```bash
# 清除可能影响连接的代理设置
unset http_proxy https_proxy
```
**作用**：确保网络连接纯净，避免代理干扰

### 第 2 步：🌐 网关启动
```bash
# 启动 OpenClaw 核心网关服务
openclaw gateway start
```
**作用**：启动后端服务，为 TUI 界面提供支持

### 第 3 步：💻 终端登录
```bash
# 进入文本用户界面
openclaw tui --password <你的密码>
```
**作用**：启动交互式终端界面，开始 AI 辅助操作

## 🎯 一键点火脚本

将以下别名添加到 `~/.bashrc`，实现一键启动：

```bash
alias claw='unset http_proxy https_proxy && openclaw gateway start && openclaw tui --password qixingovo'
```

使用方式：
```bash
# 重新加载配置
source ~/.bashrc

# 一键点火
claw
```
